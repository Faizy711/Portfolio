Bookmark Extension

Bookmark Extension is an application that makes a database copy of your Google Chrome bookmarks and organizes them in a personal library. From the library you can search for a Bookmark by its title, access a bookmark's web page, create folders, assign the folders to bookmarks and sort them accordingly.

This was the second project I worked on at the Northwestern Coding Bootcamp, it was one of my first projects that worked with a backend databse. React Drum Sequencer

The React Drum Sequencer is a 1 bar drum looper that will play a 1 bar loop according to the pattern on the pads. This is very similar to a lot of drum sequencer machines where a loop will be played and the pads or buttons triggered will make a sound. I made this project to better understand React.js and to better understand how music is programmed for simple drum machines. Many thanks to n1k0/tinysynth github repo for the general wireframe idea and also from Unlikeness's React Sequencer Tutorial for the general functionality of the sequencer. Please feel free to check it out and create your own 1 bar drum loop.
